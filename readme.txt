Portfolio plugin to showcase your portfolio items on your site

== Description ==

A powerful portfolio plugin to showcase your portfolio pictures and videos and sort them under filterable animated tabs. 
It comes with built-in animation and lighbox JQuery plugins.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'ArCo Portfolio'->'Settings' to customize Layout and JQuery options

4. Use shortcode [arco-portfolio] in your gallery page
